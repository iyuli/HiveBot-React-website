# 蜂巢官网源码

###### 预览蜂巢官网： [hiveai.cc](https://hiveai.cc/);

基于ant desing pro v2.0开发，使用项目模板：ant-design-pro-landing-page

脚手架使用的是： [antd-init](https://github.com/ant-design/antd-init);

## 我们更新了什么？

1. 我们优化并修复了，移动设备页面体验
2. 我增加了多国语言支持，自建了一套多语言系统，相比i18更加方便维护与便捷。
3. 我们优化了PC与移动端的导航栏体验。


## install
```
$ cnpm i 
//or
$ npm i
```

## 启动项目

```
$ npm start
```
## 编译项目
```
$ npm run build
```

## 目录结构

```
- locales //语言包目录
  - index.js //语言索引
  - *.json //语言包文件
- src
  - Home //页面目录
```
